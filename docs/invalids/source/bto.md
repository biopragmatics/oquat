# bto

This document details the invalid local unique identifiers used in CURIEs
for node, synonym, and definition cross-references in `bto`.


## `ISSN`: International Standard Serial Number

- Normalized prefix: `issn`
- Pattern:`^\d{4}-\d{3}[\dX]$`
