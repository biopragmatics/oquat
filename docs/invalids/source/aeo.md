# aeo

This document details the invalid local unique identifiers used in CURIEs
for node, synonym, and definition cross-references in `aeo`.


## `AEO`: Anatomical Entity Ontology

- Normalized prefix: `aeo`
- Pattern:`^\d{7}$`
