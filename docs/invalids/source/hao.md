# hao

This document details the invalid local unique identifiers used in CURIEs
for node, synonym, and definition cross-references in `hao`.


## `DOI`: Digital Object Identifier

- Normalized prefix: `doi`
- Pattern:`^(doi\:)?\d{2}\.\d{4}.*$`
