# xlmod

This document details the invalid local unique identifiers used in CURIEs
for node, synonym, and definition cross-references in `xlmod`.


## `CAS`: CAS Chemical Registry

- Normalized prefix: `cas`
- Pattern:`^\d{1,7}\-\d{2}\-\d$`
