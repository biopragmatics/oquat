# lpt

This document details the invalid local unique identifiers used in CURIEs
for node, synonym, and definition cross-references in `lpt`.

- Normalized prefix: `mgi`
- [https://bioregistry.io/mgi](https://bioregistry.io/mgi)
- Pattern:`^\d+$`
